Nuestra Metodología

Core ELO System

Adaptado del ajedrez, nuestro sistema procesa 40 años de datos ATP para asignar un valor de habilidad intrínseco a cada jugador, ajustado por superficie.

XGBoost Classifier

Utilizamos Gradient Boosting sobre 95,000 partidos para identificar patrones que el ojo humano ignora, logrando una precisión histórica del 85%.

Surface Weighting

Calculamos ratings específicos para Tierra Batida, Hierba y Pista Rápida, capturando la especialización de cada jugador.
